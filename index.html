1. What does HTML stand for?

A Hyperlinks and Text Markup Language
B Hyper Text Markup Language
C Home Tool Markup Language
D Hyper Text Making Language
HTML stands for Hyper Text Markup Language, which is the standard markup language for creating web pages.

2. Which CSS property is used to change the background color?

A color
B bgcolor
C background-color
D background
The background-color property is specifically used to set the background color of an element.

3. Which of the following is not a programming language?

A Python
B JavaScript
C HTML
D Java
HTML is a markup language used for creating web pages, but it is not a programming language as it doesn't have logic constructs.

4. Which symbol is used for comments in CSS?

A //
B /* */
C <!-- -->
D #
CSS uses /* */ to denote comments. The // syntax is used in JavaScript and other programming languages.

5. What is the latest version of HTML as of 2023?

A HTML4
B XHTML
C HTML5
D HTML6
HTML5 is the latest version of HTML, which introduced many new features and APIs for modern web development.
